Security & Hardening (DevSecOps)

Security is not a feature; it's the foundation. QELO-X implements several layers of defense-in-depth:

  • Isolated Config Tracking: Sensitive keys and RPC secrets are strictly excluded from SCM through advanced Git filtering.
  • Subprocess Jail: Node processes are strictly managed via process IDs, preventing "Zombie" processes and orphan leaks.
  • Health Score Heuristics: Real-time mathematical scoring (0-100) based on peer stability, block latency, and system IOPS.
