Seção 01

Visão Geral do Sistema

Propósito, escopo funcional e principais diferenciais
01

O QELO-X é um orquestrador de daemon para o nó go-quai da Quai Network. Evoluído de scripts bash para uma aplicação Go totalmente tipada, oferece supervisão de processo, telemetria em tempo real, dashboard web e interface terminal — tudo como um único binário daemon (qeloxd).

🔄

Auto-Environment Switch

Troca entre redes Quai com 1 clique. Reconfiguração automática de data-dir e reinício sem intervenção manual.
💯

Health Score 0–100

Índice agregado calculado em tempo real: peers, CPU, RAM, disco, estado do processo. Visualizado como círculo SVG animado.
🚨

Peer-Drop Alert

Detecção ativa de queda de conexões TCP abaixo do limiar configurável (padrão: 10), com alerta visual piscante.
🧊

Freeze Detection

Janela deslizante de altura de bloco. Sem novos blocos por mais de 10min → health score a 0 e alerta WARN.
⚙️

Backoff Exponencial

Reinícios pós-crash com delay crescente (teto 60s). Limite configurável — 0 = reinícios infinitos.
🔒

Web Basic Auth

Autenticação nativa no dashboard. CORS restrito a localhost. Hardening completo via systemd unit.
